# Generic Agent-Oriented ICM

## Purpose

This repository is a reusable Interpretable Context Methodology (ICM) framework.

It is designed to:
- isolate operational context
- improve deterministic AI execution
- support scalable multi-agent systems
- reduce prompt pollution
- provide interpretable workflows

## Instructions for AI Agents

1. Read `ICM.md`
2. Read top-level `CONTEXT.md`
3. Route into the correct workspace
4. Load the local workspace `CONTEXT.md`
5. Operate only within workspace scope

## Workspace Philosophy

- Local authority
- Explicit ownership
- Deterministic routing
- Workflow isolation
- Persistent operational state
